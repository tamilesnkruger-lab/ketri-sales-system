# PRODUCT NORTH STAR
## Sistema Comercial Ketri
Versão 1.0

## Missão

Transformar o processo de venda da Ketri em uma experiência simples, organizada e guiada, ajudando o vendedor a tomar a próxima decisão certa durante todo o relacionamento com o cliente.

O sistema deve reduzir o esforço mental do vendedor, aumentar a produtividade comercial e registrar o histórico da relação com cada cliente.

## Visão

Ser uma plataforma comercial especializada para a Ketri Criativa, desenvolvida em torno do processo real de vendas da empresa.

O objetivo não é criar um ERP genérico nem um CRM cheio de funcionalidades. O objetivo é criar a melhor ferramenta possível para vender os produtos da Ketri.

## Usuários

### Administrador

Principal dispositivo: desktop.

Objetivos:
- cadastrar produtos;
- importar produtos;
- atualizar preços;
- criar kits;
- acompanhar vendedores;
- acompanhar clientes;
- revisar orçamentos;
- analisar resultados;
- administrar o sistema.

O Admin precisa de visão ampla e ferramentas de gestão.

### Vendedor

Principal dispositivo: celular.

Objetivos:
- conversar com clientes;
- encontrar produtos rapidamente;
- montar orçamento;
- enviar mensagens;
- registrar conversa;
- criar follow-up;
- fechar vendas.

O vendedor precisa de rapidez, clareza e orientação.

## Coração do sistema

O Atendimento Guiado é o centro do Sistema Comercial Ketri.

Todos os outros módulos existem para apoiar essa jornada:

Cliente → Atendimento → Produtos → Orçamento → Registro → Follow-up → Histórico

## Princípios do Produto

### 1. Guiar sem bloquear
O sistema deve sugerir o próximo passo, mas nunca impedir o vendedor de trabalhar.

### 2. Uma decisão por vez
O vendedor nunca deve enfrentar várias decisões importantes simultaneamente.

### 3. Contexto sempre visível
Durante todo o atendimento o vendedor deve saber:
- quem é o cliente;
- em que etapa está;
- qual é o objetivo do atendimento;
- qual é a próxima ação.

### 4. Mobile First
Toda funcionalidade nova deve nascer pensando no celular e depois ser adaptada ao desktop.

### 5. Desktop otimizado para gestão
O Admin precisa de mais informações simultaneamente. O vendedor não.

### 6. O sistema pensa junto com o vendedor
O sistema não deve apenas registrar dados. Ele deve sugerir produtos, mensagens, próximos passos e follow-ups.

### 7. Produtos apoiam o atendimento
A aba Produtos existe para consulta. Ela não substitui o Atendimento Guiado.

### 8. Orçamento é consequência
O vendedor não abre o sistema apenas para criar orçamento. Ele abre para atender um cliente. O orçamento nasce naturalmente da conversa.

### 9. Histórico é memória
O vendedor nunca deve perguntar: "O que conversei com esse cliente?" O sistema deve responder isso imediatamente.

### 10. Menos é mais
Antes de criar qualquer funcionalidade nova, perguntar: isso realmente facilita a venda?

## O que o sistema NÃO é

- ERP financeiro;
- sistema de estoque completo;
- marketplace;
- catálogo online genérico;
- CRM genérico.

## O que o sistema É

- assistente comercial;
- CRM especializado;
- ferramenta de vendas;
- organizador de relacionamento;
- catálogo inteligente.

## Ordem de prioridade

1. Atendimento Guiado
2. Orçamentos
3. Produtos
4. Clientes
5. Follow-ups
6. Histórico
7. Dashboard
8. Landing Page
9. Relatórios avançados

## Critério para aprovar nova funcionalidade

Toda nova ideia deve responder "sim" às perguntas:

1. Resolve um problema real do vendedor ou do Admin?
2. Está alinhada à missão do sistema?
3. Respeita o fluxo principal de atendimento?
4. Não cria retrabalho desnecessário?
5. Pode ser implementada de forma incremental?
6. Vale os créditos do Codex?

Se alguma resposta for "não", a funcionalidade deve ir para o backlog.

## Filosofia de desenvolvimento

Antes de qualquer implementação:

1. Definir o problema.
2. Desenhar a experiência.
3. Aprovar a arquitetura.
4. Escrever a especificação.
5. Implementar.
6. Homologar.
7. Só então considerar a funcionalidade concluída.

## Frase norteadora

O Sistema Comercial Ketri é um assistente comercial inteligente que acompanha o vendedor do primeiro contato ao pós-venda, ajudando-o a tomar sempre a próxima decisão certa.
